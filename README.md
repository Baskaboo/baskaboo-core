# BASKABOO CORE

Η επίσημη βάση της Θεωρίας των Πάντων Baskaboo.

## Τι είναι η Baskaboo;
Μια θεωρία που ενώνει τα πάντα: φυσική, ψυχολογία, φιλοσοφία, μαθηματικά, θρησκεία, επιχειρήσεις, υπολογιστές — σε 4 υποπροσωπικότητες: Pits, Mits, Klop, Laram.
